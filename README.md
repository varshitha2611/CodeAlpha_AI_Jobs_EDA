# CodeAlpha_AI_Jobs_EDA
# AI Jobs Market Analysis

This project performs Exploratory Data Analysis (EDA) on AI and Data Science job market data.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Top Job Roles
- Job Category Distribution
- Salary by Experience Level
- Work Setting Analysis
- Highest Paying Countries
- Salary Trends
- Company Size vs Salary
- Employment Type Analysis
- Employee Residence Analysis
- Average Salary by Job Category

## Key Findings
- Data Engineer is the most demanded role.
- AI salaries have shown steady growth.
- Full-time employment dominates the industry.
- Machine Learning and AI roles command high salaries.
